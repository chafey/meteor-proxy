Welcome to http-server!

This is a simple http-server which does the following:
  • GET requests - returns the request headers in JSON as the body of the response
  • POST requests - returns the body in the post as the body of the response
  • PUT requests - returns HTPP status 200
  • DELETE requests - returns HTPP status 200
  • HEAD requests - returns HTPP status 200
This is convenient for testing HTTP endpoints.
